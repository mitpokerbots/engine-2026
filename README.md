# MIT Pokerbots Engine 2026
MIT Pokerbots engine and skeleton bots in Python, Java, and C++.

This is the implementation of the engine for playing this years poker variant,
where you get 3 hole cards pre-flop, and you choose a card to discard face up
after the flop.

## Dependencies
 - python>=3.5
 - cython (pip install cython)
 - Our custom package pkrbot (pip install pkrbot)
 - Java>=8 for java_skeleton
 - C++17 for cpp_skeleton
 - boost for cpp_skeleton (`sudo apt install libboost-all-dev`)

## Linting
Use pylint.
